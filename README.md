# 🌸 متجر هشّة للحلويات - Hashah Sweets Store

موقع متجر "هشّة" للحلويات والمخبوزات المنزلية الطازجة والمحضرة بكل حب وشغف يومياً. تم بناء تطبيق الويب التفاعلي هذا باستخدام **React 18** و **TypeScript** و **Vite** و **Tailwind CSS** مع حركات انتقالية تفاعلية ناعمة باستخدام **Motion**.

---

## 🛠️ دليل النشر والتشغيل السريع (GitHub & Vercel)

هذا المشروع مهيأ ومجهّز بالكامل للنشر الفوري والمباشر على منصات الاستضافة السحابية مثل **Vercel** أو **Netlify** ومستودعات **GitHub**.

### 1️⃣ الرفع إلى GitHub
لمشاركة الكود والرفع إلى مستودع جديد على GitHub، اتبع الخطوات التالية في الحزمة الطرفية (Terminal):

```bash
# تهيئة المستودع المحلي
git init

# إضافة جميع الملفات المجهزة
git add .

# تسجيل الالتزام الأول
git commit -m "feat: إطلاق متجر هشة للحلويات بأحدث التصاميم والعلب الشفافة"

# تسمية الفرع الرئيسي
git branch -M main

# ربط المستودع المحلي بمستودع الـ GitHub الخاص بك (استبدل الرابط برابط مستودعك)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# رفع الكود للفرع الرئيسي
git push -u origin main
```

---

### 2️⃣ النشر والتشغيل على Vercel 🚀

منصة **Vercel** تدعم مشاريع Vite تلقائياً وبسهولة فائقة خلال دقيقة واحدة:

1. اذهب إلى موقع [Vercel](https://vercel.com) وسجل الدخول باستخدام حساب GitHub الخاص بك.
2. انقر على **New Project** (مشروع جديد).
3. اختر المستودع الخاص بالمتجر من القائمة واضغط **Import** (استيراد).
4. ستجد الإعدادات مجهزة تلقائياً كالتالي:
   * **Framework Preset:** `Vite`
   * **Build Command:** `npm run build`
   * **Output Directory:** `dist`
5. اضغط على زر **Deploy** (نشر) 🎉.

سيبدأ الموقع بالعمل فوراً وسيزودك برابط مباشر وتحديثات فورية مع كل عملية دفع (Push) جديدة إلى GitHub!

## 💻 التشغيل المحلي والتطوير (Local Development)

إذا كنت ترغب بفتح الكود وتطويره على جهازك الشخصي:

1. تأكد من تثبيت بيئة عمل **Node.js** على جهازك.
2. قم بتثبيت الحزم البرمجية:
   ```bash
   npm install
   ```
3. ابدأ تشغيل خادم التطوير المحلي:
   ```bash
   npm run dev
   ```
4. افتح المتصفح على الرابط: `http://localhost:3000`

## ✨ مميزات المشروع التقنية والفنية
* **أداء فائق السرعة**: تم الدمج والترجمة عبر **Vite** ليعطي أفضل تجربة مستخدم وسرعة تحميل لا تضاهى.
* **إضافات سلسة للسلة**: تم تعديل الإضافة لتعطي إشعار ترحيبي راقي (**Toast Notification**) دون تحويل قسري أو مفاجئ لصفحة السلة، مما يسمح للعميل بمواصلة تصفح باقي المنتجات.
* **توافقية الصور العالية**: جميع المنتجات معروضة الآن بصور حقيقية ومطابقة تماماً لشكل العلب الشفافة الأنيقة والفاخرة المطلوبة.
* **رقم تواصل مباشر**: تم دمج رقم واتساب مباشر وآمن في الشريط السفلي وتفعيل الروابط التفاعلية.
* **متجاوب بالكامل**: واجهة استخدام وتصميم متين متوافق مع كافة أحجام الشاشات الهواتف والأجهزة اللوحية والمكتبية.
